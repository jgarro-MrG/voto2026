# Propuesta de Nuevo Cuestionario Voto2026

## Resumen del Cambio

**Antes:** 18 preguntas con escala Likert (1-5), puntuación individual por candidato
**Después:** ~10 preguntas con opciones de propuestas, puntuación compartida entre candidatos que apoyan cada propuesta

---

## Nuevo Formato de Preguntas

Cada pregunta:
1. Plantea un **problema o situación** del país
2. Ofrece **3-4 opciones** que representan enfoques diferentes
3. Cada opción tiene asociados los **candidatos que la apoyan**
4. El usuario puede seleccionar **múltiples opciones** si está de acuerdo con varias

## Sistema de Puntuación Propuesto

```typescript
// Cuando usuario selecciona una opción:
function calcularPuntuacion(opcionSeleccionada: Opcion) {
  const candidatosQueApoyan = opcionSeleccionada.candidatos;

  // Cada candidato que apoya la propuesta recibe puntos
  candidatosQueApoyan.forEach(candidato => {
    candidato.puntuacion += PESO_POR_DIMENSION;
  });
}

// Permite selección múltiple por pregunta
// Si selecciona A y C, ambos grupos de candidatos reciben puntos
```

---

## Ventajas del Nuevo Sistema

1. **Diferenciación real:** Las opciones representan enfoques genuinamente diferentes
2. **Transparencia:** Usuario ve qué candidatos apoyan cada propuesta
3. **Flexibilidad:** Puede estar de acuerdo con múltiples enfoques
4. **Reducción:** De 18 a 11 preguntas
5. **Perfil de liderazgo:** Nueva dimensión que ayuda a filtrar por estilo de gobierno
