# Cuestionario Voto2026

## Estructura del Cuestionario

Total de preguntas: **18 preguntas** (2-3 por dimensión) + **4 preguntas demográficas**

Cada pregunta tiene 5 opciones que mapean a valores de 1 a 5 en la dimensión correspondiente.

## Sistema de Puntuación

Cada respuesta genera un valor de 1-5 que se acumula por dimensión:

- Opción A = 1 punto
- Opción B = 2 puntos
- Opción C = 3 puntos
- Opción D = 4 puntos
- Opción E = 5 puntos

El perfil del usuario se calcula como el promedio de puntuaciones en las preguntas de cada dimensión.
